# Digital Salary - Контекст проекта

## Что это
Калькулятор рыночной стоимости диджитал-специалистов на основе данных SuperJob API

## Текущее состояние
- Проект на Next.js 14 + TypeScript + TailwindCSS
- Репозиторий: https://github.com/maisondina/digital-salary
- Локально запускается из: ~/digital-salary-new
- API SuperJob подключён (Secret Key в src/lib/superjob-api.ts)

## Что работает
- Интерфейс с селекторами профессии и города
- 4 профессии: Копирайтер, SMM-специалист, Таргетолог, SEO-специалист
- Карточки навыков со звёздочками (клик работает стабильно)
- Навыки адаптируются под каждую профессию
- Автопересчёт зарплаты в реальном времени
- "Вся Россия" вместо "Удалённо"
- Сбор данных через SuperJob API (с pagination, до 2000 вакансий)
- Фильтрация мусорных вакансий (агентства, без зарплаты)
- Страницы "О проекте" и "Автор"
- Код на GitHub синхронизирован

## Что можно доделать
- Учитывать опыт работы
- Показывать динамику зарплат по месяцам
- Давать рекомендации: "Выучите эти навыки для роста на +30,000 ₽"
- Попапы онбординга рядом с элементами (не в центре)

## Методология расчёта

### Сбор данных:
1. Запросить ВСЕ вакансии по профессии (с pagination, до 2000)
2. Фильтровать мусор (агентства, без зарплаты)
3. Собрать отдельно для Москвы (town=4), СПб (town=14), всей России

### Обработка зарплат:
Если from и to: salary = from + (to - from) * 0.35
Если только from: salary = from
Если только to: salary = to * 0.7

### Расчёт по регионам:
Для каждого региона: медиана(все зарплаты в регионе)
НЕ базовая * коэффициент!

### Анализ навыков:
Извлечь навыки из текста вакансий (описание + требования)
Посчитать частоту каждого
Взять топ-10 (минимум в 3+ вакансиях)
Для каждого навыка:
  medianWith = медиана(зарплаты С навыком)
  medianWithout = медиана(зарплаты БЕЗ навыка)
  salary_impact = medianWith - medianWithout
